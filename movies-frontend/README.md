# Movies Frontend | Movies Web

## Tech Stack:
- [Angular](https://angular.io/)
- [Alyle-UI](https://alyle.io/) ❤️
- [SCSS](https://sass-lang.com/)

## Installation

```bash
yarn install
```

## Running the app

```bash
# development
$ ng serve
```

## Test

```bash
# unit tests
$ ng test

# e2e tests
$ ng e2e
```

## Build

```bash
# build
ng build

# prodcution build
ng build --prod
```
