# Movies API | Movies Backend

## Tech Stack:
- [Nest.js](https://nestjs.com/)
- [Swagger](https://swagger.io/)
- [MikroORM](https://mikro-orm.io/)
- [PostgreSQL](https://www.postgresql.org/)

## Installation

```bash
yarn install
```


## Running the app

```bash
# development
$ yarn start

# watch mode
$ yarn start:dev

# production mode
$ yarn start:prod
```

## Test

```bash
# unit tests
$ yarn test

# e2e tests
$ yarn test:e2e

# test coverage
$ yarn test:cov
```
